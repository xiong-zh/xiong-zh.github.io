    Scrapy框架的使用之Item Pipeline用法

    Item Pipeline 的调用发生在 Spider 产生 Item 之后。当 Spider 解析完 Response 之后，Item 就会传递到 Item Pipeline，被定义的 Item Pipeline 组件会顺次调用，完成一连串的处理过程，比如数据清洗、存储等。
    Item Pipeline 的主要功能有如下 4 点。

    • 清理 HTML 数据。
    • 验证爬取数据，检查爬取字段。
    • 查重并丢弃重复内容。
    • 将爬取结果保存到数据库。
