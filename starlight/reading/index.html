# 读书笔记

   随着年龄的增长，感觉时间过的越来越快，也越加觉得知道的东西太少，思维受限，迫切的需要拓展知识面，了解各个方面的东西，阅读是个不错的方式，从去年开始就有了制定阅读清单并去执行的这一想法， 也就施行了，18年读了一些书,愈发觉得不断补充所需的重要性,计划会一直执行下去,希望19年好好多读几本书,充实自己.    这些书为2019年必读和需要去读的书,多为技术需要和大牛好评的书.部分书籍已看过,但考虑其掌握程度和技术需要还会认真的去多度几遍, 阅读原则为技术书求精不求多,其他书籍求多不求精,多多益善,加粗为今年需要去精读的书.
